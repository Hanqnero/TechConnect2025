Главный экран
	календарь с датами
		при клике на дату:
			пользователь видит занятия на которые он записан на эту дату
			учитель видит список занятий которые он ведет
				при клике на занятие открывается экран "отметить на занятии"
	если ученик посещал занятие в дату она выделена цветом
				
	при логине ученик попадает на главный экран ученика
	при логине учитель попадает на главный экран учителя

	кнопка профиль, переводит на экран профиля
		
профиль ученика
	список секций на которые записан ученик, есть кнопка "Отписаться от секции" которая отписывает студента от секции
	ниже список из всех секций, есть кнопка "Записаться на секцию", которая записывает студента на секцию

профиль учителя
	список секции к которым есть доступ на редактирование
	кликнув на секцию открывается экран редактирования секции
		
экран "Отметить на занятии"
	список записанных секцию учеников
	можно отметить галочкой студентов, и отправить
	отправка создает посещение для выбранных студентов

		
экран редактирования секции
	можно посмотреть список учеников в секции
	
Логика работы секций со стороны студента:
	Студент может записать на секцию на экране профиля
	У секции есть график занятий
	Студент видит в календаре даты с будущими занятиями своих секций (можно их пометить обводкой)
	Студент видит занятия на которых его отметили (они заливаются зеленым)
